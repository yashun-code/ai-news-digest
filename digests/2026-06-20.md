## Claude Code に大型アップデート（レート上限2倍 ＋ Artifacts ＋ クラウド実行）
Claude Code の使用上限が**2倍**になりました。さらに、
- **Artifacts**：作業の成果を「その場で更新される共有Webページ」にできる（手順書・ダッシュボード・チェックリスト向け）
- **クラウド実行のルーティン**：自分のMacを起動してなくても、決めた時間に自動で作業が走る
- GitHubのコードレビューが設定不要で使える

- term: Artifacts(アーティファクト)＝作った成果物を見える形で残す機能／ルーティン＝定期自動実行
- relate: 決まった時間に情報収集や定例作業を自動で走らせられるようになります。「毎週まとめを作る」といった作業の自動化に向いています。
- source: https://releasebot.io/updates/anthropic
- source: https://releasebot.io/updates/anthropic/claude-code

## OpenAI、Codex を ChatGPT に統合（週500万人突破・Record & Replay 追加）
OpenAIのCodexが ChatGPT に組み込まれ、役割別プラグインや、操作を録画するだけで再利用できる手順に変える **Record & Replay**（6/18）が追加。週500万人が使い、うち2割が一般のビジネス職になりました。

- term: Codex＝OpenAIのコーディング支援AI。Claude Codeの競合／Record & Replay＝やって見せた作業を覚えて自動化
- relate: Claude Code と Codex を「自動で長く働く相棒」として使い分ける時代に入りつつあります。
- source: https://releasebot.io/updates/openai/codex
- source: https://www.digitalapplied.com/blog/openai-codex-chatgpt-business-plugins-june-2026-agentic-work

## Google、Gemini 3.1 と「企業向けエージェント基盤」を発表
Googleが最新モデル **Gemini 3.1 Ultra** を公開。あわせて、Salesforce・ServiceNow・Workday などの業務ソフトとつながる **Gemini Enterprise Agent Platform**（自律的に仕事をこなすAIを会社の道具と連携させる基盤）を発表しました。

- term: エージェントAI＝指示の先まで自分で考えて作業を進めるAI
- relate: AIが「会話の相手」から「会社の業務をこなす働き手」へ。業務効率化や新しいサービスのアイデア源になります。
- source: https://en.wikipedia.org/wiki/Gemini_Enterprise_Agent_Platform

## xAI の Grok、複数のAIを「同時並行」で走らせる機能
イーロン・マスク氏のxAIが、1つの作業に対して**複数のAIを同時に動かせる**ようにしました。これまでは1つ終わるまで待つ必要があったものが、並行で進められます。

- term: 並列エージェント＝複数のAIを同時に走らせて作業を分担・高速化する仕組み
- relate: 「AIに何でも1人でやらせる」から「AIをチームで動かす」へ。役割を分けて複数のAIを並行で使う流れが広がっています。
- source: https://releasebot.io/updates/xai

## Anthropic、ソウルに拠点を開設し韓国のAI連携を強化
Anthropic（Claudeの開発元）が**ソウルにオフィスを開設**し、韓国のAI関連企業・機関との新たな提携を発表（6/17）。AI企業が各国に拠点を置き、地域ごとに連携を深める動きが続いています。

- relate: AI企業と国・地域の連携が標準になりつつある、という規制・信頼性まわりの動き。
- source: https://releasebot.io/updates/anthropic
