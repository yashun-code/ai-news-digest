## Anthropic、Claude Design を機能強化し Claude Code との連携開始
Anthropic が「Claude Design」の機能を大幅刷新。デザインシステムのインポート機能、コード の往復編集対応、トークン消費の問題を解決するなど、デザイナーと開発者の連携をスムーズにしました。Claude Code との統合により、デザイン→実装 の一気通貫が可能になります。

## OpenAI の Codex に「Record & Replay」機能、画面操作を自動化
Codex が新機能「Record & Replay」を追加。ユーザーが6秒の画面操作を録画すると、AI が約25秒で自動完成させる仕組み。YouTube アップロードなど複雑な作業の代行が可能に。開発者向けAIツールの競争が加速しています。

- term: Record & Replay＝人間の画面操作を記録して、AIが続きを自動生成する機能
