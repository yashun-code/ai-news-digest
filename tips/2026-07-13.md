## Claude Code とは
Claude Code はAnthropicが提供するデスクトップアプリケーションで、AI が直接あなたのコンピュータでコード作成や開発作業をサポートします。最新版では Web ブラウザが統合され、AI が外部サイトを閲覧・操作できるようになりました。

- term: Claude Code＝デスクトップアプリで AI が開発を手伝うツール
- source: https://docs.claude.com/ja/docs/claude-code

## ブラウザ機能で何ができるのか
Claude Code の新しいブラウザ機能を使うと、AI が外部 Web サイトにアクセスしたり、リンクをクリック、フォームに入力したりできます。これにより、データ収集やテスト自動化の効率が大幅に向上します。
