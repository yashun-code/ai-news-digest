## 7つの指示方法を知ろう
Claude Codeへの指示には、目的に応じた複数のアプローチがあります。単純な機能追加から複雑なプロジェクト設計まで、「何をしたいのか」を明確に伝える方法を選ぶことで、AI が正確に理解しやすくなります。今回は、公式が解説する7つの方法を、初心者向けに整理します。

## 具体的な依頼文を作る
「〇〇を作ってください」だけでなく、使い道や条件を追加すると、Claude Code はより役立つコードを生成します。たとえば「ユーザーが数値を入力できる計算機が必要」「エラー対応も含める」など、実装の目的や制約を明確にします。

- kind: skill
- term: 具体的な指示＝「何を」「どのように」「なぜ」を一文に込める
- try: Claude Codeに「入力値の検証機能付きのメモアプリをPythonで作成してください」と指示してください
- source: https://docs.claude.com/ja/docs/claude-code

## 段階的に指示を重ねる
最初に基本的なコード骨組みを生成してから、「エラー処理を追加」「UIを改善」と段階的に指示することで、細かい調整がしやすくなります。一度のやり取りで完璧を目指さず、反復的に改善するのがコツです。

- kind: skill
- term: 反復指示＝小さな要件を積み重ねてコードを育てる方法
- source: https://docs.claude.com/ja/docs/claude-code

## 計画モードで全体像を共有する
複雑なプロジェクトでは、実装に入る前に「こんなアプリを作りたい、機能は〇〇と△△」と全体像を説明することが重要です。Claude Code が目標を理解していれば、各ステップで一貫性のあるコードが生成されます。

- term: 計画モード＝実装前に要件や設計思想を共有する

- source: https://docs.claude.com/ja/docs/claude-code
