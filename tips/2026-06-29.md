## React フックとは
React フックは、クラスを書かずに状態を管理できる仕組みです。最も使われるのが `useState` で、変数の値を保存して、変更するたびに画面を自動で更新します。フックは `use` で始まる関数で、コンポーネントの中だけで使えます。

- kind: lesson
- term: フック＝React の関数で、コンポーネントに機能を追加するツール
- source: https://docs.claude.com/ja/docs/claude-code

## useState で状態を作る
`useState` は 2 つの値を返します：現在の値と、その値を変える関数です。括弧の中に初期値を入れます。ボタンを押すたびに数字が増えるカウンターを作ってみましょう。

```jsx
import { useState } from 'react';

export default function Counter() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>カウント: {count}</p>
      <button onClick={() => setCount(count + 1)}>増やす</button>
    </div>
  );
}
```

- term: useState＝状態を管理する基本的なフック。値を保存して変更できる

## useEffect で副作用を処理する
`useEffect` は、画面が描画されたあとに何か実行したいときに使います。たとえば、API からデータを読み込んだり、タイマーを設定したりできます。

```jsx
import { useState, useEffect } from 'react';

export default function Timer() {
  const [time, setTime] = useState(0);

  useEffect(() => {
    const timer = setInterval(() => setTime(t => t + 1), 1000);
    return () => clearInterval(timer);
  }, []);

  return <p>経過時間: {time} 秒</p>;
}
```

- kind: skill
- term: useEffect＝コンポーネントが描画されたあとに処理を実行するフック
- try: このタイマーコンポーネントを試して、秒数が増えていくのを確認してください
- source: https://docs.claude.com/ja/docs/claude-code

## フックのルール
フックは必ずコンポーネントのトップレベルで呼び出します。ループやif文の中には書きません。また、フックは公開されたコンポーネントの中だけで使える特別な機能です。
