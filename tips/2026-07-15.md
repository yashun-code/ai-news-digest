## useState フックで状態を作る
状態とは、コンポーネントが覚えておく値のこと。`useState` を使うと、ボタンをクリックするたびにカウントが増えるなど、変わる値を扱えます。

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>カウント: {count}</p>
      <button onClick={() => setCount(count + 1)}>増やす</button>
    </div>
  );
}
```

- kind: skill
- term: useState = 関数コンポーネント内で状態を宣言する React の基本フック
- term: count = 現在の状態の値
- term: setCount = 状態を更新する関数
- try: useState を使ったシンプルなカウンター機能を実装してください

## useEffect フックで副作用を実行する
副作用とは、画面を更新した後に実行する処理のこと。APIからデータを取得したり、タイマーを設定したりします。`useEffect` はコンポーネント表示時に1回だけ実行できます。

```jsx
import { useEffect, useState } from 'react';

function DataFetcher() {
  const [data, setData] = useState(null);
  
  useEffect(() => {
    // ページ表示時に1回だけ実行
    fetch('https://api.example.com/data')
      .then(res => res.json())
      .then(json => setData(json));
  }, []); // 空配列 = 1回だけ実行
  
  return <div>{data ? JSON.stringify(data) : 'ロード中...'}</div>;
}
```

- kind: skill
- term: useEffect = コンポーネントのライフサイクルで副作用を実行するフック
- term: 依存配列 = 第2引数。空配列なら1回だけ、省略すると毎回実行
- try: API から天気データを取得し、useEffect で表示するコンポーネントを作成してください
- source: https://docs.claude.com/ja/docs/claude-code

## フックの3つのルール
フックを正しく使うための約束があります。必ず守ってください。

1. **トップレベルで呼ぶ** = if文やループの中で呼ばない
2. **関数コンポーネントから呼ぶ** = 通常の関数では呼べない
3. **useで始まる** = React のルール

フックを正しく使うことで、コンポーネントは安定して動作します。
